# [fit] Systematic Debugging

---

# [fit] `breakpoint()` is like a pause button in your Python code. 

---

# [fit] 
1. Guess where you think the problem is.
2. Insert a `breakpoint()` there.
3. Check your assumptions.
4. Make one change in the code.
5. See if it works.
6. Rinse & repeat.
